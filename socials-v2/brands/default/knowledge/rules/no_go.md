# Regler & Begränsningar

## Saker vi INTE får säga eller lova
Lista claims och formuleringar som är juridiskt eller varumärkesmässigt problematiska.

- Vi får inte lova garanterade resultat som "du får fler kunder", "vi dubblar din försäljning" eller "du rankar etta på Google"
- Vi får inte använda absoluta påståenden som "bäst", "ledande", "marknadens främsta" eller liknande utan tydligt belägg
- Vi får inte kalla lösningar certifierade, officiella eller godkända om det inte faktiskt stämmer
- Vi får inte lova exakta tidsramar eller affärseffekter som vi inte säkert kan hålla i varje projekt
- Vi får inte antyda att AI, automation eller digitala lösningar löser allt utan begränsningar
- Vi får inte använda vilseledande formuleringar kring pris, leverans eller omfattning
- Vi får inte lova juridiska, ekonomiska eller tekniska resultat som ligger utanför vårt faktiska uppdrag

## Känsliga ämnen att undvika
- Jämförelser med konkurrenter vid namn
- Nedvärderande formuleringar om andra byråer, leverantörer eller kunders tidigare lösningar
- Politiska ställningstaganden eller partipolitiska budskap
- Kontroversiella samhällsfrågor som inte är relevanta för vår kärnverksamhet
- Kundspecifik information, interna detaljer eller case som inte är godkända att delas offentligt

## Varumärkesskyddade termer
Lista om det finns termer, namn eller koncept ni äger eller INTE får använda.

- Coastworks och Coastworks.se ska användas konsekvent som varumärke
- Vi ska vara försiktiga med att använda andra företags varumärken, produktnamn eller logotyper utan tillstånd
- Vi får inte ge intryck av officiella samarbeten, partnerskap eller certifieringar om sådana inte finns
- Eventuella kundnamn, plattformar eller tredjepartsverktyg ska bara nämnas när det är relevant och korrekt

## Specifika no-go-ord
- billigast
- snabbast
- störst
- garanterat
- revolutionerande
- game-changer
- disruptiv
- magisk
- viral
- världsklass
- bäst i klassen
